# 📝 TODO Web App

A simple and lightweight TODO web application built using **HTML, CSS, and JavaScript**.
This project helps users manage daily tasks in an easy and minimal way.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* ⚡ Fast and works directly in the browser (no installation needed)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Setup

1. Clone the repository:

```
https://github.com/sneha-mankala/Todo-Application.git
```

2. Open the project folder:

```
cd todo-webpage
```

3. Run the project:

* Simply open `index.html` in your browser
